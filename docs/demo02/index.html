「安心」のあるホーム。
便利さと静けさをバランスよく備えた東京・山の手の邸宅街、南平台…。

 

南平台は、麻布、白金台、松濤などと並ぶ東京・山の手の中でも指折りの邸宅街。代官山や恵比寿と隣り合い、ちょっと足をのばせば銀座や日本橋も近い、抜群の立地です。

ご家族や友人も気軽に訪れやすいロケーションと、いつも見守られている安心と、上質なくつろぎを運ぶ住空間の演出。トラストガーデン南平台では人生の集大成を迎えるにふさわしい住環境とサービスをご用意しております。
