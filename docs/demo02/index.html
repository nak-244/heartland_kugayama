高齢者がペットと暮らす効用
  • ペットと暮らしていない人と比べて血圧が低く、ペットに話しかける人は血圧の効果作用が認められた。
  • 犬の世話をする人は、そうでない人と比較して医者にかかる回数が21％減少した。
  • ペットと暮らす人は鬱病にかかるリスクが減る。
  • ペットの話をすることで、友人が多くできやすい。
  • ペットを散歩に連れて行ったりと、より活動的になる。
  • ペットを良き友としてみなす。
  • ペットの存在が夫や妻を失ったストレスを軽減する。
  • 寂しさを減少する。
  • ペットをケアすることで、自分自身の体も大切にする。
  • 信頼しているペットがいることで、日常生活において安心・安全の感情が芽生える。
